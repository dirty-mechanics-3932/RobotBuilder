CAN Talon

What is it?

A Talon speed controller that uses CAN protocol to control the the attached motor. In addition to controlling speed, Talons controlled by CAN can do PID and uses limit switches to control motion.

Properties

CAN ID
The ID of the CAN device. This must match the ID of the device configured with BDC-COMM and shown in the roboRIO WebDash.
